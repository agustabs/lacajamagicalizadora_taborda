Proyecto para modulo de ReactJs en Coderhouse.
El objetivo del proyecto es la implementación de un eCommerce funcional.
En este caso elegí llevar a cabo un eCommerce de vinos para un futuro proyecto ya en marcha dónde se pueda acceder a catalogos de vinos curados.
La idea final sería que el mismo pueda contener todo el catalogo, y la opción de generar una orden de compra completa que luego pueda ser abonada en paralelo, por ejemplo con MP o TRANSFER.
